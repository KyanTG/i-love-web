Werkt bij GSAP

Animatie bibliotheek die je van alles laat animeren, van simpele dingen tot meer complex zoals 3d dingen.

De diepte van GSAP is bizar, elke keer als je er naar kijkt zie je weer nieuwe mogelijkheden

je kan bijvoorbeeld met GSAP makkelijk diverse animatie timelines laten starten, een stuk makkelijker dan met CSS.

een van de populairste gebruikte functies is de scroll triggers

een van de minder populaire gebruikte functies is de utils bibliotheek

prefers-reduced-motion is handig om te gebruiken zodat het toegankelijker is voor mensen die snel duizelig worden van animaties.

met control methods kan je dingen animeren zoals met pauze / reversal button, maar ook met bijvoorbeeld een scroll bar om de animatie te versnellen.

helper functions kan je gebruiken om problemen op te lossen van klanten / jezelf

met container animation kan je bijvoorbeeld een pagina maken waarin een container horizontaal scrollt inplaats van verticaal

met ease visualizer kan je aparte eases maken waardoor je bijvoorbeeld een bliksem animatie kan maken

met setmaps / utils.mapRange kan je animaties bepalen afhankelijk van schermgrootte waardoor animaties werken op elk grootte scherm

met matchmedia() gebruik je GSAP versie van media queries CSS
